Refactorización Lista de Pacientes ECH (Versión Full Clinical)
Actúa como un Senior UI Engineer especializado en sistemas EHR. Tu objetivo es crear un sistema de listas modular, de alta densidad y 100% alineado con nuestra arquitectura de Angular 15.

1. Creación de la Lógica Modular (layout_list.js)
Crea un archivo llamado layout_list.js que gestione la inyección dinámica de los componentes de navegación de la lista.

header-component: Mantén la cabecera actual con el título central en 2rem (.fs-lg / .header-title) y el logo.

tab-bar-component: Inyecta la franja de pestañas (Arrivals Today, Planned Arrivals, etc.) con fondo grisáceo y botones de configuración (icono pi-cog).

toolbar-component: Inyecta la barra de herramientas ampliada que incluya:

Paginación a la izquierda (<< < 1 > >>).

Grupo de iconos de acción en el centro-derecha (Pdf, Filtro, Carpeta).

Buscador con contador azul (badge-count) a la derecha.

2. Estilo de Alta Densidad y Tipografía (theme-overrides.css)
Base: Mantén html { font-size: 13px; }.

Escala: Asegura que el texto general sea de 1.3rem y los títulos de 2.0rem.

Tabla Clínica:

Crea un contenedor .table-scroll-container que permita scroll vertical y horizontal (overflow: auto).

La tabla debe tener un min-width: 1500px para evitar que las columnas se amontonen en pantallas de 14''.

Añade estilos para los encabezados (<th>) que incluyan un efecto hover (fondo gris claro) e iconos de ordenación de PrimeNG (pi-sort-alt).

3. Estructura de la Tabla en el HTML
Refactoriza la página de lista principal para que use esta estructura de columnas:

Habitación (Negrita, azul).

Admisión (Icono de color: Rojo para Urgencias, Azul para Hospitalizado).

Paciente (Nombre en negrita, con Rec ID y Episodio debajo en fuente pequeña).

Edad/Sexo (Usa iconos pi-venus / pi-mars).

Problema Médico (Texto truncado con el diagnóstico principal).

Médico Responsable (En azul negrita).

Payer (Seguro).

Alertas (Usa p-tag de PrimeNG con iconos: DNR, Alergia, etc.).

Días de Estancia (Número resaltado).

Status (Iconos compactos de Medicación, Órdenes y Vitals con indicador de "nuevo" / dot-new).

Acciones (Icono vertical pi-ellipsis-v al final de cada fila).

4. Documentación Técnica
Añade comentarios indicando que el archivo layout_list.js facilita la creación de nuevas listas (ej. Lista de Altas, Lista de UCI) simplemente cambiando el contenido de las pestañas.

Documenta que los iconos de ordenación son marcadores de posición para la lógica de sort de PrimeNG.
